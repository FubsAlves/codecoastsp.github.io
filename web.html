HTML & CSS.

O que é HTML?

HTML (abreviação para a expressão HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web. HTML5 é a quinta versão da linguagem HTML, que adicionou mudanças relacionadas a semântica e acessibilidade.

O que é CSS?

Cascading Style Sheets (CSS) é um simples mecanismo para adicionar estilo (cores, fontes, espaçamento etc) a um documento HTML. Por facilidade em vez de se colocar a formatação dentro do documento, o CSS cria um link (ligação) para uma página que contém os estilos. Quando quiser alterar a aparência do site basta portanto modificar o arquivo.
